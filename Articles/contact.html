Feedback

We welcome any suggestions for improving the Magic Library. Feel free to offer any constructive comments that you feel would enhance future revisions. These may range from minor changes for improving ease-of-use to new major areas of functionality. Please also feel free to comment on how code could be improved using alternative techniques.

Email: feedback@dotnetmagic.com

Bug Reports

Please report any bugs found whilst using the library no matter how trivial they may seem. It is tempting to assume that an obvious problem must have already been notified by other users but this is often not the case. If possible give the version number of the library along with as much detail as possible about how to reproduce the problem. Often a problem occurs easily for an end user but proves very difficult to reproduce on another system.

Email: bugs@dotnetmagic.com

Website problems

If you have problems accessing any part of the website or notice any broken links then please notify the webmaster as soon as possible so it can be rectified.

Email: webmaster@dotnetmagic.com
